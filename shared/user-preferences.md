# User Preferences

Fine-grained preferences for how the AI team should interact with you. These override defaults.

> **Note**: These are sensible defaults. Your AI will discover and refine your preferences during your first few conversations. You can also edit this file directly at any time.

## Language & Formatting
- **Primary language**: English
- **Secondary language**: None
- **Date format**: YYYY-MM-DD
- **Currency**: USD
- **Time zone**: UTC

## Response Style
- **Default length**: Medium (1-2 paragraphs)
- **Use headers in responses**: Only for long responses
- **Use bullet points**: Frequently
- **Include emoji**: Never
- **Formality level**: Professional

## Workflow Preferences
- **Auto-save outputs**: Yes — always save to the active system's `C-creations/`
- **Review pipeline**: Ask me each time
- **Proactive suggestions**: Yes — suggest next steps
- **Memory updates**: Auto-log to the active system's `I-insights/`

## Notification Preferences
- **Progress updates**: After each step
- **Confirmation needed for**: Only external actions
