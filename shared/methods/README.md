# Shared Methods

This directory contains reusable methods, SOPs, and protocols that apply across all your systems.

## How to Add a Method

Create a new `.md` file in this directory with the following structure:

```markdown
# Method: [Name]

## Purpose
[What this method accomplishes]

## When to Use
[Trigger conditions — when should the AI suggest this method?]

## Steps
1. [Step 1]
2. [Step 2]
3. [Step 3]

## Expected Output
[What the result should look like]

## Notes
[Any caveats, tips, or context]
```

## Available Methods

| Method | File | Purpose |
|--------|------|---------|
| Competitive Analysis | `competitive-analysis.md` | Systematically evaluate competitors to find strategic opportunities |
| Content Repurposing | `content-repurposing.md` | Transform one piece of content into multiple formats for different platforms |
| Decision Framework | `decision-framework.md` | Make better decisions faster by structuring the thought process |
