# Creations

This directory stores deliverables and outputs created by your AI agents.

## What Goes Here

- Drafted content (blog posts, emails, social posts)
- Research reports and analysis documents
- Generated presentations and proposals
- Any other AI-produced deliverables

## Organization

Files are organized by type and date. You can create subdirectories to keep things tidy:

```
C-creations/
├── blog-posts/
├── reports/
├── emails/
└── social/
```
