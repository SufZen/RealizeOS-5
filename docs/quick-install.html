RealizeOS Setup

Welcome to the easiest installation guide ever.

1

Download Python

RealizeOS relies on Python to run its AI features. If you already have Python 3.11 or higher installed, you can skip this step!

If not, go to the official Python website, download Python 3.12+, and crucially, make sure you check the box that says "Add Python to PATH" during installation.

2

Install Node.js (for Dashboard)

The RealizeOS dashboard is a modern React app that needs Node.js 18+ to build. This is optional — the backend API works without it.

Download and install Node.js LTS from the official website. The installer handles everything automatically.

3

Extract RealizeOS

Find the RealizeOS folder you downloaded (likely an extracted .zip file on your computer).

Simply extract the contents anywhere on your hard drive. Your Desktop or Documents folder is perfectly fine!

4

Configure API Keys

RealizeOS needs at least one LLM provider API key to function. You have two options:

Option A (Recommended): Open a terminal in the RealizeOS folder and run:

python cli.py setup

This interactive wizard will guide you through configuration, install dependencies, and build the dashboard.

Option B (Manual): Copy .env.example to .env and fill in your API keys with a text editor.

5

Launch the System

Everything is ready. We designed a simple one-click launcher for you.

Open your extracted RealizeOS folder and double-click the file named:

start-realizeos.bat

(The very first time you click this, it may take 1-2 minutes to automatically download standard AI dependencies in the background. Please be patient!)

💡 Tip: You can also run python cli.py serve from a terminal for more control.
✓

You're All Set!

The system is now running locally on your computer. When you launch the .bat file, RealizeOS will load an icon into your system tray and serve the dashboard directly on your local browser.

You can access the dashboard manually any time at:

http://localhost:8080
📋 Useful commands:
python cli.py doctor — diagnose your installation
python cli.py setup — re-run the setup wizard
